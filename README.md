# Wazuh VM Lab Project

## Project Overview
This project demonstrates the deployment of a Wazuh Manager inside a virtual machine hosted on a Windows system. The goal was to monitor endpoint activity and enforce security policies using the Wazuh SIEM platform.

## Objectives
- Deploy Wazuh Manager in a VM environment
- Connect endpoints as agents
- Configure security policies
- Monitor logs and alerts

## Lab Environment

Host Machine:
- Windows 10

Virtualization:
- VirtualBox

Security Platform:
- Wazuh Manager

## Architecture
Windows Host → Virtual Machine → Wazuh Manager → Endpoint Agents

## Setup Process

### Step 1
Installed VirtualBox on the Windows host.

### Step 2
Created a Linux virtual machine for Wazuh Manager.

### Step 3
Installed Wazuh Manager inside the VM.

### Step 4
Connected endpoint agents.

### Step 5
Configured security policies for endpoint monitoring.

## Key Features Demonstrated
- SIEM deployment
- Endpoint monitoring
- Log collection
- Security alert detection
- Policy configuration
